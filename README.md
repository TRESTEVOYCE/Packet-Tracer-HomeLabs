# Cisco Networking Homelabs Portfolio

Welcome to my networking homelab portfolio repository.

This repository contains Cisco Packet Tracer (`.pkt`) projects and network simulations that I have designed and configured to demonstrate my knowledge and hands-on skills in network design, configuration, troubleshooting, and maintenance.

These labs simulate real-world enterprise network environments and showcase my understanding of Cisco networking concepts based on CCNA-level technologies.

##  Repository Contents

This repository includes:

- Cisco Packet Tracer network topologies
- VLAN and network segmentation projects
- Router and switch configurations
- IP addressing and subnetting designs
- Inter-VLAN routing implementations
- Routing protocol configurations
- Network security implementations
- Troubleshooting scenarios
